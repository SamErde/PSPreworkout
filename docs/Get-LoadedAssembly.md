# Get-LoadedAssembly

## SYNOPSIS
Get all assemblies loaded in PowerShell.

## SYNTAX

```
Get-LoadedAssembly [<CommonParameters>]
```

## DESCRIPTION
Get all assemblies loaded in PowerShell.

## EXAMPLES

### EXAMPLE 1
```
Get-LoadedAssembly
```

Returns a list of all loaded assemblies.
