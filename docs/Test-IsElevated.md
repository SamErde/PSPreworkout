# Test-IsElevated

## SYNOPSIS
Check if you are running an elevated shell with administrator or root privileges.

## SYNTAX

```
Test-IsElevated [<CommonParameters>]
```

## DESCRIPTION
Check if you are running an elevated shell with administrator or root privileges.
