# Edit-WingetSettingsFile

## SYNOPSIS
Edit the WinGet settings file.

## SYNTAX

```
Edit-WingetSettingsFile [<CommonParameters>]
```

## DESCRIPTION
A shortcut to edit the WinGet settings file.
This will create one if it does not already exist.

## EXAMPLES

### EXAMPLE 1
```
Edit-WinGetSettingsFile
```
